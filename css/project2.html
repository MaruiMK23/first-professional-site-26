Project Overview

Custom-built arcade stick controller designed for competitive gaming and fighting game enthusiasts.

Project Details

  • Category: Gaming Hardware
  • Client: Gaming Community
  • Date: February 2024
  • Service: Hardware Design

Description

This custom arcade stick features 18 illuminated neon green buttons, a small OLED display for input monitoring, and a translucent acrylic top panel. Designed for competitive fighting games, it includes directional inputs on the left side and action buttons on the right, all connected via USB for PC and console compatibility.
